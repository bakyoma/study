처음 git 사용방법

1. 올리고자 하는 폴더 주소창에서 "cmd" 를 입력한다
2. git init을 입력한다
3. git add . (올리고자 하는 모든 폴더 전체) 를 입력한다
4. git commit -m "메모" 를 입력한다
5. git branch -M main 를 입력한다
6. git remote add origin "올리고자 하는 주소" 를 입력한다
7. git push -u origin main (확정) 을 입력한다

수정/추가된 파일 올릴때

1. 올리고자 하는 폴더 주소창에서 "cmd" 를 입력한다
2. git add . 을 입력한다
3. git commit -m "메모" 를 입력한다
4. git push 를 입력한다
