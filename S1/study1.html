처음 git 사용방법

1. 올리고자 하는 폴더 주소창에서 "cmd" 를 입력한다
2. git init을 입력한다
3. git add . (올리고자 하는 모든 폴더 전체) 를 입력한다
4. git commit -m "메모" 를 입력한다
5. git branch -M main 를 입력한다
6. git remote add origin "올리고자 하는 주소" 를 입력한다
7. git push -u origin main (확정) 을 입력한다

수정/추가된 파일 올릴때

0. git pull 을 입력해서 파일을 다운을 받는다 (협업할때)
1. 올리고자 하는 폴더 주소창에서 "cmd" 를 입력한다
2. git add . 을 입력한다
3. git commit -m "메모" 를 입력한다
4. git push 를 입력한다

새로운 html 파일 만드는 법

1. ctrl+n 을 입력한다 (파일이 생성)
2. ctrl+shift+s 를 입력한다 "파일명".html 을 입력한다
3. 명령어 "!" 를 입력한다

새로운 컴퓨터에서 작업물을 불러올때, 오류나서 새로 받을때

1. 불러오고자 하는 폴더 주소창에 "cmd"를 입력한다
2. git clone "주소창(github code 주소 복사)"을 입력한다
